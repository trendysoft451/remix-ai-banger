# À personnaliser avant la production

Le dépôt est techniquement prêt pour GitHub et Render, mais aucun contrat public vérifiable de l'API « Remix Banger AI » n'a été fourni. Avant de désactiver `MOCK_MODE`, vous devez renseigner les URL et la clé du fournisseur réel, puis adapter le mapping de ses réponses si nécessaire.

Vous devez également remplacer dans `privacy-policy.html` :

- la durée de conservation ;
- les sous-traitants réels ;
- votre identité légale ;
- votre adresse de contact.
